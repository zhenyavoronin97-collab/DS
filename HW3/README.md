План:
Выбор датасета и подготовка окружения в Colab (установка PostgreSQL, библиотек).
Настройка Kaggle API и загрузка датасета.
Анализ структуры датасета с помощью pandas (df.info()).
Создание таблицы в PostgreSQL с соответствующими типами данных (INTEGER, TEXT, FLOAT, DATE).
Импорт данных из датасета в PostgreSQL через df.to_sql().
Написание SQL-запросов с фильтрацией (WHERE), агрегатными функциями (COUNT, AVG, SUM, MAX, MIN) и группировкой.
Визуализация данных: построение различных графиков (линейные, столбчатые, гистограммы, ящики с усами, тепловая карта корреляции) с использованием matplotlib и seaborn.
Исправление предупреждений seaborn (palette + hue).
Обсуждение различных стилей оформления графиков.
