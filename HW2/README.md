Проект для анализа данных о продажах компании. Позволяет загружать данные из CSV, очищать их, фильтровать по датам, агрегировать по категориям и строить наглядные графики.



- `data\_loader.py` — загрузка данных из CSV-файла.
- `data\_processer.py` — очистка, фильтрация и агрегация данных.
- `visualisation.py` — визуализация (столбчатые диаграммы, точечные графики, временные ряды).
- `main.py` — основной скрипт, объединяющий все этапы анализа.
