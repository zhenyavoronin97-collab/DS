1.Загрузка и первичный анализ данных:
Загрузка файла AirQualityUCI.csv.
Объединение колонок Date и Time в индекс DateTime.
Очистка от полностью пустых столбцов и строк с некорректными датами.
Описание признаков и их характеристик.
2.Разведочный анализ (EDA):
Визуализация пропусков, заполнение пропусков методом forward fill.
Графики временных рядов основных загрязнителей.
Гистограммы распределений, корреляционная матрица.
Проверка стационарности (ADF-тест) и декомпозиция ряда (тренд, сезонность).
3.Предобработка данных для машинного обучения:
Создание лаговых признаков (lag_1, lag_2, lag_3, lag_24) и скользящих средних.
Добавление временных признаков (час, день недели, месяц) из индекса.
Удаление строк с NaN.
Масштабирование признаков и целевой переменной (MinMaxScaler).
Разделение на обучающую (80%) и тестовую (20%) выборки с сохранением временного порядка.
4.Построение и обучение моделей прогнозирования (5+ подходов):
SARIMA (одномерная модель для CO с суточной сезонностью).
LSTM (рекуррентная нейросеть с последовательностями длины 24).
CNN (одномерная сверточная сеть).
Transformer (модель на базе механизма внимания).
XGBoost (градиентный бустинг для регрессии).
Для каждой модели вычисляются метрики MAE и RMSE.
5.Сравнение моделей:
Построение столбчатых диаграмм MAE и RMSE.
Формирование сводной таблицы метрик.
6.Поиск аномалий с помощью автоэнкодера:
Подготовка последовательностей (окна по 24 часа).
Обучение сверточного автоэнкодера на обучающих данных.
Вычисление ошибки реконструкции на тесте и определение порога аномалии (95-й процентиль).
Визуализация обнаруженных аномалий и примеры аномальных окон.
7.Визуализация результатов:
Графики прогнозов всех моделей на тестовой выборке.
Остатки (ошибки) лучшей модели (Transformer).
Распределение ошибок.

Важность признаков на примере XGBoost (столбчатая диаграмма).
